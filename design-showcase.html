Fluid Typography

No breakpoints needed — type scales smoothly between minimum and maximum sizes using CSS clamp(). Resize the browser to see it flow.

clamp() fluid

Body text stays at 1rem — readability is already optimal at this size across all viewports. Only headings and display text need fluid scaling.

const fluid = clamp(min, preferred, max);
/* Fluid type — no breakpoints needed */
font-size: clamp(2.25rem, 5vw + 0.5rem, 4.5rem);

/* How it works: */
/* min: 2.25rem (36px) — phone floor */
/* pref: 5vw + 0.5rem — scales with vw */
/* max: 4.5rem (72px) — desktop ceiling */
02

Fluid Spacing

Spacing tokens also use clamp() — tighter on mobile, generous on desktop. No jarring jumps at breakpoints.

--space-xs
clamp(0.25rem, 0.5vw, 0.5rem)
--space-sm
clamp(0.5rem, 1vw, 0.75rem)
--space-md
clamp(0.75rem, 1.5vw + 0.25rem, 1.5rem)
--space-lg
clamp(1.25rem, 2.5vw + 0.25rem, 2.5rem)
--space-xl
clamp(2rem, 4vw + 0.5rem, 5rem)
--space-2xl
clamp(3rem, 6vw + 1rem, 8rem)
/* Page padding adapts fluidly */
--page-pad: clamp(1rem, 4vw, 2.5rem);

/* Sections breathe with the viewport */
margin-top: var(--space-xl);
padding-top: var(--space-lg);
