# Tools

## Communication

- **Telegram** is the primary channel. All messages come through Telegram DM.

## Voice Transcription

Voice messages are automatically transcribed using OpenAI Whisper.
When a user sends a voice message in Telegram, the `openai-whisper-api` skill
converts it to text so you can read and respond to it.

## Web Search

You can search the web using `web_search` (Brave Search API) and fetch page
content with `web_fetch`. Use these to look up information for the user.

## Managed Skills

- `skills/nbhd-managed/weekly-review/` — End-of-week synthesis
